使用 GitHub

fork & pull request

  • fork 操作相当于把别人的 git 仓库克隆到自己账号中。
  • pull request 则可以将自己 fork 过来的仓库中的修改合并到原仓库中，实现团队协作。

第 3/3 步：添加 NAME 内容是 GitHub 账号名。
